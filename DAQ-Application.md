DAQ = 把各种传感器信号（光、电、温度、振动等）实时采集 → 变成数字数据 → 分析系统性能。
光学系统里，DAQ就是 “所有测试数据的入口”。

| 用途      | 说明           |
| ------- | ------------ |
| 系统测试    | 测试光学系统是否达到指标 |
| 环境测试    | 振动、温度对光学影响   |
| 飞行/地面遥测 | 记录系统运行状态     |


DAQ采集的信号

| 传感器               | 采的数据  |
| ----------------- | ----- |
| Photodiode        | 光功率   |
| CCD / IR detector | 图像信号  |
| Thermocouple      | 温度    |
| Strain gauge      | 结构形变  |
| Accelerometer     | 振动    |
| Voltage monitor   | 电源稳定性 |
